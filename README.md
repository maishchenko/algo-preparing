# Подготовка к алгоритмическому интервью (Яндекс)

Репозиторий содержит задачи и план подготовки к алгоритмическому собеседованию в Яндекс.

## Структура

- `leetcode_problems.md` — список 87 задач с LeetCode, сгруппированных по темам
- `leetcode_plan.md` — 10-дневный план подготовки: 58 задач, ~6 задач в день
- `problems/` — решения задач на Python
